What's Next?

This project represents a very basic attempt at using digital tools for solving complex historical problems. Going forward, the following suggestions might be useful,

1. I did not record the hadith numbers along with the edge attributes. Recording such information, while adds more time to data collection phase, will be even more productive in tracing exact hadith as they pass down from one transmitter to another. 

2. Unfortunately, my network diagrams did not use arrows to show directed edges, even though this information was embedded in the data. This is a small oversight that can be easily corrected in the future. 

3. Advanced mapping tools will also aid us in improving our mapping abilities. A mapping project of Islamic historical geography is in progress, which hopefully will allow easier access to understanding geographical networks. 

4. Lastly, the more data we have, the more refined our conclusions will be. Data collections remains the most challenging part for a project like this. With the development of machine readable Arabic texts, this issue can be fixed in the future. Once we develop a system of digital data collection, our possibilities will expand beyond our imaginations for this field of study.
